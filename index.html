    • Common-utils

    It is a small package with some useful classes for easy and quick java desktop development that can be used for whatever common project, contains classes and interfaces for Window objects(Swing package based) and handlers (listeners) for some common events(ActionListener,MouseListeners,MouseMotionListener,MouseWheelListener) also have classes to build 2D and 3D Graphics almost from Scratch(Just using native Graphics2D from JVM).
